 BurpSuite Setup Guide

            
    
              
  1. Open BurpSuite → Proxy → Options
    2. 
              
  2. Ensure listener is on 127.0.0.1:8080 (or your port)
    3. 
              
  3. Enable "Intercept" in the Proxy tab to capture requests
    4. 
              
  4. Import BurpSuite CA certificate to avoid SSL errors
    5. 
              
  5. Enable proxy above and click Send — requests will appear in BurpSuite
    6. 
            

            

              
Quick BurpSuite CA Certificate Import:

              curl -x http://127.0.0.1:8080 http://burp/cert -o burp_ca.der
